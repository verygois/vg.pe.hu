bastian void / ant'lrd japan tour 2017

アメリカから来日する二人の音楽家、bastian void (Joseph Bastardo/Looks Realistic/Gay Shapes) と ant'lrd (Baro Records/Inner Island/etc..)が、東京・金沢・京都・大阪にて国内外の仲間たちと共演、計6つの公演をお送りします。

まずは東京にて、10月31日から11月2日迄の3デイズ。ツアー初日となる10月31日(火)は、同じく同時期に日本をツアーする二人のカナダ人アンビエント音楽家、Daniel MajerとJonathan William Scherk(ジョナサンはこの日が誕生日！)が競演。加えて、アメリカツアーの際にはbastian voidの家に泊めさせてもらっているというほどの仲良し、食品まつり a.k.a Foodman (Orange Milk/Digitalis/etc..) がソロ演奏 ＆ bastian voidとのセッションを予定。

続く11月2日(木)東京と11月5日(日)金沢公演には、昨年bastian voidが来日した際にも共演し、大阪を拠点とする音楽レーベル「birdFriend」から bastian void との共作アルバムを発表した盟友H. Takahashi(Where To Now?/Unknown Me)の出演が決定。

11月8日(水)京都公演には、bastian voidが運営するカセットレーベル「Moss Archive」から新作をリリースした奈良在住のカナダ人音楽家Endurance、ツアー最終日11月10日(金)大阪公演(11月11日はant'lrd の誕生日！)には、同じく「Moss Archive」からアルバムをリリースしたTakahiro Mukai (Phinery/birdFriend/etc...)、昨年ジョーさんが来日した際にも共演を果たした7FO (RVNG Intl/Bokeh Versions)が出演します。

その他にも各公演に、bastian void と ant'lrd がぜひ会いたいと熱望した開催地近くに住む音楽家を加え、国境を越えてつながる現行の音楽シーンを象徴する素敵なラインナップが揃いました。どうぞご期待ください。

  • Bastian Void

    バスチャン・ヴォイドは、1986年生まれ、マサチューセッツ州出身のグラフィックデザイナー・作曲家として活躍するジョセフ・バスタードが手掛ける音楽プロジェクト。2009年にLooks Realisticでのバンド活動を始め、2011年に初めてのBastian Voidとしてのアルバム「Ported」をリリース。それ以降、バスチャン・ヴォイドなどの名義で、Orange Milk、birdFriend、Digitalis、Constellation Tatsu、Phineryなどのレーベルで数多くのアルバムをリリース。作曲の際、アナログ、デジタル、モジュラーシンセサイザーを使用。Moss Archiveというレーベルを経営している。
